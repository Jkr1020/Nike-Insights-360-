
#  Nike Sales Dashboard – Power BI

##  Project Overview

The **Nike Sales Dashboard** is an interactive data visualization project built using **Microsoft Power BI**.
It provides clear insights into Nike’s sales performance across different regions, products, and time periods.
The dashboard helps stakeholders analyze trends, identify high-performing products, and support data-driven decision-making.

---

##  Objectives

* Analyze overall sales performance of Nike products
* Identify top-performing regions and products
* Track revenue trends over time
* Provide interactive filtering for better analysis

---

##  Dataset Description

The dataset used in this project contains sales-related information, including:

* **Order ID** – Unique identifier for each sale
* **Order Date** – Date of transaction
* **Region** – Sales region
* **Product / Category** – Nike product details
* **Units Sold** – Quantity sold
* **Revenue / Sales Amount** – Total sales value

> *The dataset is cleaned and transformed using Power Query before visualization.*

---

##  Dashboard Features

* **Total Sales & Revenue KPIs**
* **Sales by Region (Bar / Map Visuals)**
* **Product-wise Sales Analysis**
* **Monthly & Yearly Sales Trends**
* **Interactive Filters (Slicers)** for:

  * Region
  * Product Category
  * Time Period

---

## Tools & Technologies Used

* **Microsoft Power BI** – Dashboard creation & visualization
* **Power Query** – Data cleaning & transformation
* **DAX (Data Analysis Expressions)** – Measures and calculated fields

---

##  Project File

* **Sales Dashboard Nike.pbix**
  *(Power BI project file containing data model, visuals, and calculations)*

---

##  How to Use the Dashboard

1. Download and install **Power BI Desktop**
2. Open the file: `Sales Dashboard Nike.pbix`
3. Use slicers to filter by region, product, or date
4. Analyze insights through interactive visuals

---

##  Key Insights (Example)

* Certain regions contribute the highest share of total revenue
* Seasonal trends impact sales performance
* A few product categories generate the majority of sales
